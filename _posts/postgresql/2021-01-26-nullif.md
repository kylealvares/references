The `NULLIF` function returns `NULL` if the two expressions are equal or the first expression if not.

### Syntax

```https
SELECT NULLIF(<expression-one>, <expresion-two>);
```

### Example #1

```https
SELECT NULLIF(2, 9);
```

### Example #2

```https
SELECT 10 / NULLIF(2, 9) 
AS nullif;
```

### Example #3

```https
SELECT COALESCE(10 / NULLIF(0, 0), 0) 
AS nullif;
```
