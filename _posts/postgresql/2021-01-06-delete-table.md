The ```DROP TABLE``` command is used to create a table in SQL.

### Syntax

{% highlight sql %}
DROP TABLE <table-name>;
{% endhighlight %}

### Example

{% highlight sql %}
DROP TABLE Companies;
{% endhighlight %}
