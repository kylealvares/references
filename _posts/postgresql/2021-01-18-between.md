The ```BETWEEN``` operator selects values within a specified range.

### Syntax

{% highlight sql %}
SELECT <columns>
FROM <table-name>
WHERE <column-name> BETWEEN <value-1> AND <value-2>;
{% endhighlight %}

### Example

{% highlight sql %}
SELECT *
FROM Companies
WHERE founded BETWEEN 1920 AND 1960;
{% endhighlight %}

--------+---------------+---------+---------------
 Disney | Entertainment |    1923 | United States
