The ```CREATE DATABASE``` command is used to create a database in SQL.

### Syntax

{% highlight sql %}
CREATE DATABASE <database-name>;
{% endhighlight %}

### Example

{% highlight sql %}
CREATE DATABASE Company;
{% endhighlight %}
