The ```DROP DATABASE``` command is used to create a database in SQL.

### Syntax

{% highlight sql %}
DROP DATABASE <database-name>;
{% endhighlight %}

### Example

{% highlight sql %}
DROP DATABASE Company;
{% endhighlight %}
